Chapter 1 Licensing R

Licensing for a package which might be distributed is an important but potentially complex subject.

Writing R Extensions

1.1 Disclaimer

This book is by no mean a legal book and should not be used as such.

This book aims at helping you decipher the complexity of open source licenses, but if you have legal concerns and questions about open source licenses, please refer to a professional lawyer.

1.2 What is this book about

This book aims at two things:

  • Documenting the current state of licensing in the R world

  • Summarize and explain technical terms specific to open source license.
